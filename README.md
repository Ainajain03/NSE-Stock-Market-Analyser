# 📈 NSE Stock Market Analyser

An AI-powered stock market analysis platform focused on NSE-listed companies that combines fundamental analysis, technical analysis, valuation metrics, market sentiment, and financial visualization into a single decision-support system.

## 🚀 Project Overview

NSE Stock Market Analyser was developed to help investors move beyond price speculation and make data-driven investment decisions.

The platform analyzes listed Indian companies using:

- Fundamental Analysis
- Technical Analysis
- Valuation Analysis
- Financial Statement Analysis
- NSE Holding Pattern Analysis
- Sector Comparison
- Growth Metrics
- Risk Assessment
- AI-generated Insights

The system converts raw financial data into actionable insights for retail investors, students, analysts, and long-term wealth creators.


# 🎯 Problem Statement

Most retail investors rely on:

- Social media tips
- News headlines
- Influencer recommendations
- Price movements alone

This often leads to emotional decision-making.

The objective of this project is to create a unified platform that automatically evaluates stocks using quantitative and qualitative parameters and presents easy-to-understand conclusions.
